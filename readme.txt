This plugin will add functionality in your site to add/edit/remove events and display them in a widget.

== Description ==

This plugin will add a custom post type for event. In that you will be able to add/edit/remove events just as you do to your
posts. There are 9 parameters are by default supported. These are-

*   Event date
*   Event Duration
*   Offical Website
*   City
*   State
*   Country
*   Event Gallery
*   Event Categories
*   Event Tags

And also you can add title, description, featured image to your event.

In the widget you can specify the title, maximum height of the widget, number of events to show and to display event from a
category or tag.

Event state, city and countries are included in the search parameter. That means if someone search anything that matches with these
meta values, they will show up in the search results.


== Installation ==

Installation of this plugin is very simple.

1. Download and unzip the "nom-event-management.zip" file.
2. Upload the "nom-event-management" folder to the `/wp-content/plugins/` directory
3. Activate the plugin through the 'Plugins' menu in WordPress
And Voila! Now you have the events menu item in your menu in the dash board and also a widget.
